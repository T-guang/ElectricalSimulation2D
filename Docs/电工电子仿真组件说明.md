# 电工电子仿真组件说明

本文档用于说明当前 Unity 2D 电工电子仿真 Demo 中的家庭电路组件和工业电路组件。当前版本以教学演示和老板亮相为目标，仿真逻辑采用简化模型：电源提供相线/零线，开关类组件控制通断，负载类组件在形成回路后显示运行状态。

## 一、家庭电路组件

### 220V电源

用于提供家庭电路中的火线 L 和零线 N。它是家庭照明、插座、电风扇、电灯泡等组件的供电来源。

当前仿真中，L 端作为相线供电端，N 端作为零线回路端。负载同时接入 L 和 N 后，才会被判断为形成完整回路。

### 交流电源

用于展示三相五线制交流电源，包含 L1、L2、L3、N、PE 五个端点。

L1、L2、L3 表示三相相线，N 表示零线，PE 表示保护地线。当前版本中，它可以作为更完整的电源模型进行接线演示，也方便后续扩展三相电机、工业控制电路等内容。

### 单开单控开关

用于控制一条线路的通断，常见于一个开关控制一盏灯的家庭照明场景。

当前仿真中，双击开关可以切换 ON/OFF。ON 时输入端和输出端导通，OFF 时断开。

### 单开双控开关

用于双控照明场景，例如楼梯上、下两个位置都可以控制同一盏灯。

当前版本中，它作为双控开关的基础演示组件，具有一个公共端和两个输出端。后续可以进一步完善为两个双控开关配合控制同一负载的教学电路。

### 电灯泡(220V)

家庭电路中的照明负载。

当电灯泡的火线端和零线端形成完整回路时，仿真状态显示 RUN，表示灯泡被点亮。它适合用于最基础的照明控制演示。

### 电风扇(220V)

家庭电路中的电器负载。

工作逻辑与灯泡类似：火线端和零线端接通并形成回路后，电风扇显示 RUN。它可以用于演示多个负载并联、开关控制负载等教学内容。

### 单相电能表(220V)

用于模拟家庭入户电路中的电能计量装置。

常见接线包含 L 进、L 出、N 进、N 出。当前仿真中，电能表内部会将进线和出线按对连接，用于表示电源经过电能表后继续供电。

### 空气开关1P

用于单极线路保护，通常控制一条相线的通断。

当前版本中，它作为可开关通断组件使用。双击可切换 ON/OFF，ON 时进线和出线导通。

### 空气开关2P

用于同时控制火线和零线，常见于家庭总开关或回路保护。

当前仿真中，2P 空开有两组进出端子。ON 时两组端子分别导通，OFF 时断开。适合做家庭照明示例的主开关。

### 空气开关3P

用于三相线路的通断控制。

当前版本中，它可用于三相供电或工业电路的简化演示。ON 时三组端子分别导通。

### 空气开关4P

用于三相四线或三相五线系统中的多极通断控制。

当前仿真中，它主要用于展示多极开关结构，适合后续扩展三相电源、N 线联动保护等场景。

## 二、工业电路组件

### 熔断器(FU)

用于电路保护。当线路发生过载或短路时，真实熔断器会熔断，切断电路。

当前 Demo 中，熔断器作为可导通保护元件使用。它可以放在主回路或控制回路中，用于表现电源进入负载前需要经过保护器件。

### 复合开关(SB)

复合按钮通常同时包含常开触点和常闭触点，常用于启动、停止、控制切换等回路。

当前版本将其作为可切换控制器件，带有多组端点，适合演示按钮控制线圈、指示灯或电磁阀。

### 自锁开关(SB)

自锁开关按下后可以保持状态，常用于需要保持通电状态的控制场景。

当前仿真中，双击可切换 ON/OFF，用于表现保持型开关的通断效果。

### 限位开关(复合)

限位开关用于检测机械位置，到达指定位置后改变触点状态。

复合型限位开关可以同时提供常开和常闭触点，常用于自动往返、电机行程控制等电路。

### 限位开关(自锁)

用于位置检测并保持动作状态的简化演示。

当前版本可作为行程控制器件参与接线，用于后续实现自动往返电动机控制电路。

### 刀开关(QS)

用于主电源隔离和手动通断控制，常见于工业电路主回路。

当前仿真中，刀开关具有三相进出端子。ON 时 L1/T1、L2/T2、L3/T3 分别导通，适合作为三相主回路的总开关。

### 交流接触器(KM)(220V)

接触器用于通过线圈控制主触点吸合，常用于电机启动、停止和正反转控制。

当前 Demo 中，220V 接触器主要作为线圈负载展示。当 A1 和 A2 接入完整回路后，显示 RUN，表示线圈得电。后续可扩展为线圈得电后联动主触点和辅助触点。

### 交流接触器(KM)(380V)

功能与 220V 接触器类似，但线圈或主回路额定电压为 380V。

当前版本用于工业三相控制电路的展示，适合自动往返、电机正反转等后续图纸。

### 三相异步电动机(380V)

工业电路中最常见的三相负载。

当前版本提供 U、V、W 和 PE 等端点，用于三相接线演示。现阶段仿真为简化负载显示，后续可增强为必须接入三相电源后才运行。

### 星三角电机(380V)

用于星三角启动控制的电机模型。

当前版本提供多端点接线基础，适合后续制作星形启动、三角运行的教学电路。

### 热继电器(FR)(380V)

热继电器用于电机过载保护。

真实电路中，热继电器检测主回路电流，过载时断开控制回路。当前 Demo 中作为三相保护通断元件使用，后续可以加入过载触发逻辑。

### 电磁阀(24V)

用于控制气动或液压回路中的阀门动作。

当前仿真中，它作为 24V 线圈负载。A1/A2 形成回路后显示 RUN，表示电磁阀得电动作。

### 通电延时时间继电器(KT)(220V)

用于通电后延时动作的控制回路。

当前版本作为线圈负载展示，接通后显示 RUN。后续可以扩展为延时若干秒后改变触点状态。

### 通电延时时间继电器(KT)(380V)

功能与 220V 通电延时时间继电器类似，但额定电压为 380V。

适合用于工业控制回路的延时启动、顺序控制等教学场景。

### 断电延时时间继电器(KT)(220V)

用于断电后延迟释放的控制回路。

当前版本先作为基础线圈负载，后续可以扩展断电保持计时和触点延迟恢复逻辑。

### 断电延时时间继电器(KT)(380V)

功能与 220V 断电延时时间继电器类似，但额定电压为 380V。

适合用于电机停机延时、设备顺序断电等教学演示。

### 开关电源(220V)

用于将交流 220V 转换为低压直流电源，例如 24V。

当前 Demo 中，它既可以作为输入电源组件，也提供 24V 输出端点，方便后续演示 PLC、传感器、电磁阀等低压控制回路。

### PLC输出模块(24V)

用于模拟 PLC 的数字量输出模块。

当前版本提供 Y 点和 COM 端点，用于展示 PLC 输出控制继电器、电磁阀、指示灯等负载。后续可扩展为点击输出点切换状态。

### 6位端子竖排

用于整理和中转线路。

端子排本身不产生电源，也不作为负载，主要用于让线路结构更清晰。当前版本作为可导通端子组件使用。

### 3位端子竖排

用于少量线路的中转和整理。

适合控制回路、小型电路或局部接线教学。

### 2位端子竖排

用于最简单的两路线路连接或中转。

适合演示端子排的基础作用。

### 6位端子横排

用于横向布置的多位端子连接。

适合在画布中做横向接线整理，减少线路交叉。

### 3位端子横排

用于小型横向端子连接。

适合控制回路中三组信号线或电源线的中转。

### 2位端子横排

用于简单的横向双端子连接。

适合做电源输入、负载输出等简化演示。

### 步进驱动器(24V)

用于控制步进电机运行，通常接收 PUL、DIR、ENA 等控制信号。

当前 Demo 中提供控制信号端和电源端，主要用于展示步进控制系统的接线关系。后续可扩展为脉冲控制和方向控制动画。

### 步进电机(24V)

用于根据驱动器输出脉冲转动的电机。

当前版本作为低压电机负载展示，提供 A+/A-/B+/B- 等线圈端点，便于和步进驱动器连接。

### 红色指示灯(220V)

用于显示设备状态，例如电源、故障、运行等。

当前仿真中，形成完整回路后显示 RUN，表示指示灯点亮。

### 绿色指示灯(220V)

常用于显示正常运行、启动完成或允许操作状态。

当前版本作为 220V 指示负载使用。

### 黄色指示灯(220V)

常用于显示警告、待机、准备或中间状态。

当前版本作为 220V 指示负载使用。

### 红色指示灯(380V)

用于 380V 回路中的状态显示。

当前版本作为工业高压回路指示负载，适合三相控制电路演示。

### 绿色指示灯(380V)

用于 380V 回路中的运行或正常状态显示。

当前版本作为工业指示负载使用。

### 黄色指示灯(380V)

用于 380V 回路中的警告或中间状态显示。

当前版本作为工业指示负载使用。

## 三、端点和接线说明

每个组件的端点都是可点击的连接点。真实图片或模型只负责显示外观，线路连接依赖端点对象。

常见端点含义如下：

- L、L1、L2、L3：相线或火线端。
- N：零线端。
- PE：保护地线端。
- A1、A2：线圈两端。
- IN、OUT：输入端和输出端。
- COM：公共端。
- Y0、Y1、Y2：PLC 输出点。

当前系统中，端点位置采用相对坐标保存，例如 `(0.5, 1.0)` 表示组件顶部中央。以后替换真实模型时，只需要把端点坐标对准图片上的接线孔中心，原有接线和仿真逻辑就可以继续使用。

## 四、当前仿真简化规则

1. 电源组件提供相线和零线。
2. 开关、空开、熔断器、刀开关等组件用于控制线路通断。
3. 灯泡、风扇、电机、线圈、指示灯等组件作为负载。
4. 负载同时接入相线和零线后，显示 RUN。
5. 如果相线和零线直接连通，系统提示短路。
6. 当前版本以演示和教学为主，复杂逻辑如接触器辅助触点、时间继电器延时、热继电器过载保护、PLC 输出控制等，可以在后续版本逐步增强。
