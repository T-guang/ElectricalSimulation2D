# 电工数字学生仿真系统 使用说明书

软件类型：Unity 2D PC 端电工电子仿真软件。

当前定位：演示版、教学练习版。

适用对象：中专、大专电工电子基础教学，适合课堂展示、老板演示、学生基础接线练习。

## 一、软件运行思路

本软件不是工业级精确电气仿真软件，而是一个面向教学和演示的 2D 电路仿真系统。

软件核心思路是：

1. 将电气元件做成可以拖动的 2D 模型。
2. 每个元件上设置若干端点。
3. 用户点击端点进行接线。
4. 软件根据端点之间的连接关系形成电路网络。
5. 点击开始仿真后，程序判断哪些元件得电。
6. 得电元件显示运行状态，例如灯泡 RUN、风扇 RUN、电机 RUN、接触器吸合。

简化理解：

用户看到的是电路图和元件，软件内部判断的是端点之间有没有导通。

## 二、软件主页面

软件目前主要包含五个入口：

1. 模拟电路。
2. 图纸集。
3. 仿真广场。
4. 元件百科。
5. 个人中心。

当前重点开发的是：

1. 模拟电路。
2. 图纸集。
3. 元件百科。

仿真广场和个人中心可以先作为空页面保留，后期再扩展。

## 三、模拟电路页面

模拟电路页面是软件最核心的页面。

页面主要区域包括：

1. 顶部工具栏。
2. 左侧电工控件池。
3. 中间画布。
4. 画布上的元件和导线。
5. 底部或侧边状态提示。

### 1. 顶部工具栏

顶部工具栏主要功能：

1. 开始仿真。
2. 删除所有线。
3. 清空画布。
4. 设置导线颜色。
5. 保存图纸。
6. 导入图纸。
7. 撤销。
8. 重做。
9. 锁定画布。

### 2. 左侧电工控件池

左侧控件池用于选择元件。

目前分为两大类：

1. 家庭电路组件。
2. 工业电路组件。

每个元件可以点击或拖入画布，放置后成为一个可操作的电路元件。

### 3. 中间画布

中间画布用于摆放元件和接线。

画布具有网格背景，便于观察元件位置和导线走向。

画布中的主要操作：

1. 放置元件。
2. 拖动元件。
3. 点击端点接线。
4. 选中元件。
5. 删除元件。
6. 选中导线。
7. 删除导线。
8. 开始仿真。

## 四、元件系统

软件中的每个元件都由三部分组成：

1. 元件外观。
2. 元件端点。
3. 元件逻辑。

### 1. 元件外观

元件外观可以是：

1. 简单色块。
2. PNG 图片。
3. 截图风格模型。
4. 后期绘制的真实元件模型。

如果以后有真实元件模型，只需要将 PNG 图片导入 Unity，并配置对应端点位置即可。

### 2. 元件端点

端点是接线的核心。

一个元件可以有多个端点，例如：

1. 220V 电源有 L 和 N。
2. 灯泡有 L 和 N。
3. 单开双控开关有 L、L1、L2。
4. 三相交流电源有 L1、L2、L3、N、PE。
5. 接触器有 L1、L2、L3、T1、T2、T3、A1、A2。

端点有自己的类型和颜色。

常见端点颜色：

1. L 或火线：红色。
2. N 或零线：蓝色。
3. PE 或保护地：绿色。
4. 三相 L1/L2/L3：可使用黄、绿、红等颜色区分。

### 3. 元件逻辑

元件逻辑用于判断元件是否导通或运行。

例如：

1. 开关闭合后，输入端和输出端导通。
2. 灯泡同时接到 L 和 N 后亮。
3. 风扇同时接到 L 和 N 后运行。
4. 接触器线圈 A1/A2 得电后，主触点闭合。
5. 三相电机接到三相电源后运行。

## 五、连线系统

连线系统用于连接不同元件的端点。

### 1. 基本接线方法

接线步骤：

1. 点击第一个端点。
2. 鼠标移动时出现预览线。
3. 点击第二个端点。
4. 软件生成完整导线。

如果点错了，可以点击空白处取消接线。

### 2. 导线颜色规则

导线颜色根据端点类型自动判断。

规则如下：

1. 接到火线端点时，导线显示红色。
2. 接到零线端点时，导线显示蓝色。
3. 接到 PE 端点时，导线显示绿色。
4. 工业三相端点根据端点颜色显示对应颜色。

### 3. 导线显示规则

导线采用直角折线显示。

当前目标：

1. 尽量减少线与线重叠。
2. 尽量避免导线穿过元件主体。
3. 导线在端点附近进入元件。
4. 导线整体保持清晰可读。

### 4. 删除导线

删除导线步骤：

1. 点击选中导线。
2. 按 D 或 Delete。
3. 导线被删除。

也可以使用“删除所有线”按钮清空全部导线。

## 六、仿真逻辑

软件当前采用简化仿真逻辑。

它不计算真实电压、电流、电阻、功率，而是判断：

1. 电源是否存在。
2. 开关是否闭合。
3. 线路是否连通。
4. 负载是否同时接到所需电源端。
5. 控制元件是否满足动作条件。

### 1. 家庭电路仿真

家庭电路主要判断 L 和 N 是否形成完整回路。

例如灯泡：

1. 灯泡 L 端接到火线。
2. 灯泡 N 端接到零线。
3. 中间开关处于闭合状态。
4. 灯泡显示 RUN。

例如风扇：

1. 风扇 L 端接到火线。
2. 风扇 N 端接到零线。
3. 中间开关或空开导通。
4. 风扇显示 RUN。

### 2. 单开单控开关

单开单控开关用于控制一个负载。

常见接线：

1. 电源 L 接开关 L。
2. 开关 L1 接灯泡 L。
3. 电源 N 接灯泡 N。
4. 开关闭合时灯亮。
5. 开关断开时灯灭。

### 3. 单开双控开关

单开双控常用于两个位置控制同一个灯。

常见接线：

1. 电源 L 接第一个双控开关的 L。
2. 第一个双控开关的 L1 接第二个双控开关的 L1。
3. 第一个双控开关的 L2 接第二个双控开关的 L2。
4. 第二个双控开关的 L 接灯泡 L。
5. 电源 N 接灯泡 N。
6. 两个开关状态相同时，线路导通。
7. 两个开关状态不同时，线路断开。

注意：双控开关不是“任意一个开关 ON 灯就亮”，而是两个开关通过两根联络线切换通路。

### 4. 工业电路仿真

工业电路目前优先实现基础逻辑。

主要包括：

1. 按钮导通。
2. 接触器线圈得电。
3. 接触器主触点闭合。
4. 电机得电运行。
5. 指示灯得电亮起。

例如接触器：

1. A1 接到控制电源火线。
2. A2 接到控制电源零线。
3. 控制回路导通后，接触器吸合。
4. 接触器吸合后，L1/T1、L2/T2、L3/T3 主触点导通。

## 七、图纸集

图纸集用于展示和选择练习图纸。

### 1. 图纸大类

图纸分为：

1. 工业电路图纸。
2. 家庭电路图纸。

### 2. 难度分类

图纸难度分为：

1. 初级图纸。
2. 中级图纸。
3. 高级图纸。

### 3. 图纸使用方式

使用步骤：

1. 点击顶部“图纸集”。
2. 选择工业图纸或家庭图纸。
3. 选择全部、初级、中级或高级。
4. 点击图纸卡片。
5. 查看大图预览。
6. 点击进入电路配置。
7. 图纸作为参考图进入模拟电路场景。

### 4. 图纸参考功能

图纸进入模拟电路场景后：

1. 可以缩小显示。
2. 可以拖动位置。
3. 可以放大查看细节。
4. 可以关闭。
5. 不参与自动布线。

图纸只是参考，不会自动生成元件和导线。

## 八、元件百科

元件百科用于说明元件作用和端点用途。

当前分类包括：

1. 开关按钮。
2. 保护设备。
3. 仪表电源。
4. 用电设备。
5. 控制设备。
6. 端子与模块。

百科页面适合学生查看：

1. 这个元件是什么。
2. 这个元件用在哪里。
3. 这个元件有哪些端点。
4. 端点大概怎么接。

## 十一、常见操作说明

### 1. 添加元件

1. 在左侧组件池找到元件。
2. 点击或拖动元件到画布。
3. 元件出现在画布上。

### 2. 移动元件

1. 鼠标按住元件。
2. 拖动到目标位置。
3. 松开鼠标。
4. 相关导线会跟随刷新。

### 3. 删除元件

1. 点击元件。
2. 元件出现选中状态。
3. 按 D 或 Delete。
4. 元件和相关导线被删除。

### 4. 连接导线

1. 点击一个端点。
2. 移动鼠标查看预览线。
3. 点击另一个端点。
4. 导线生成。

### 5. 删除导线

1. 点击导线。
2. 按 D 或 Delete。
3. 导线删除。

### 6. 开始仿真

1. 完成元件接线。
2. 点击“开始仿真”。
3. 查看灯泡、风扇、电机、指示灯等状态。

### 7. 清空画布

点击“清空画布”后，所有元件和导线都会被清除。

### 8. 删除所有线

点击“删除所有线”后，只删除导线，保留元件。

### 9. 保存图纸

点击“保存图纸”后，当前画布的元件、位置、状态、导线关系会保存。

### 10. 导入图纸

点击“导入图纸”后，可以恢复之前保存的图纸。

## 十二、推荐演示流程

给老板演示时建议按下面顺序：

1. 打开软件，展示顶部导航。
2. 进入模拟电路页面。
3. 从左侧拖入 220V 电源、单开单控开关、灯泡。
4. 接线：电源 L 到开关 L，开关 L1 到灯泡 L，电源 N 到灯泡 N。
5. 点击开始仿真，展示灯泡亮灭。
6. 再展示双控开关控制灯泡。
7. 切换到工业组件，展示接触器、电机、按钮。
8. 简单搭建电机控制电路。
9. 切换到图纸集，展示工业图纸和家庭图纸分类。
10. 点击一张图纸进入配置，展示图纸参考图可以移动和缩放。
11. 切换到元件百科，展示元件说明。

这个流程能体现软件的完整性：

1. 有界面。
2. 有元件。
3. 能接线。
4. 能仿真。
5. 有图纸。
6. 有百科。
7. 有教学价值。

## 十四、小白学习路线

如果是初学者，可以按下面顺序学习：

1. 认识电源 L、N、PE。
2. 学会连接一个灯泡。
3. 学会使用单开单控开关。
4. 学会使用单开双控开关。
5. 学会使用空气开关。
6. 学会看电表接线。
7. 认识三相电源 L1、L2、L3。
8. 认识接触器 A1/A2 和主触点。
9. 学会电机点动控制。
10. 学会电机自锁控制。
11. 学会正反转控制的基本思路。
12. 学会看图纸并按图接线。

## 十五、总结

本软件第一版的核心价值不是代替专业仿真软件，而是让学生和演示对象快速理解电路接线逻辑。

第一版最重要的目标是：

1. 能看懂。
2. 能拖动。
3. 能接线。
4. 能仿真。
5. 能按图练习。

只要这五点稳定，软件就已经具备演示价值和教学价值。
