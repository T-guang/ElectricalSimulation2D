# ElectricalSimulation2D 使用说明

项目路径：`E:\Projects\Unity\ElectricalSimulation2D`

## 打开方式

1. 打开 Unity Hub。
2. 选择 `打开/添加项目`。
3. 选择 `E:\Projects\Unity\ElectricalSimulation2D`。
4. 推荐使用 Unity `2022.3.57f1c1` 打开。
5. 打开场景：`Assets/Scenes/Demo.unity`。
6. 点击运行 Play。

如果场景不存在或需要重新生成：

- Unity 顶部菜单选择 `Tools/Electrical Demo/Build Demo Scene`。

## 当前第一版功能

- 左侧电工控件池。
- 可放置元件：220V 电源、单开开关、电灯泡、电风扇、空气开关、熔断器、按钮、接触器线圈、电机。
- 中间网格画布。
- 从左侧拖拽元件到画布。
- 点击元件端子进行接线。
- 支持直角线和直线两种导线样式。
- 支持导线颜色切换。
- 双击开关、按钮、空气开关、熔断器可以切换 ON/OFF。
- 点击“开始仿真”进行简单导通判断。
- 灯泡、风扇、电机、接触器线圈在 L/N 回路完整时显示 RUN/动作状态。
- 检测火线和零线直接连通的短路情况。
- 支持保存图纸和导入图纸，保存路径在 Unity 的 `Application.persistentDataPath` 下。

## 演示建议

建议第一场演示做“单开控制电灯”：

1. 放置 `220V电源`、`空气开关1P`、`单开开关`、`电灯泡`。
2. 电源 L 接空气开关进线。
3. 空气开关出线接单开开关进线。
4. 单开开关出线接电灯泡 L。
5. 电灯泡 N 接电源 N。
6. 双击空气开关和单开开关，使其显示 ON。
7. 点击“开始仿真”，电灯泡显示 RUN/亮起状态。

## 设计边界

这是教学演示型仿真，不是工业级电气求解软件。当前逻辑只判断导通关系、短路和基础负载动作，不计算真实电压、电流、功率、相位或电机转矩。
